# arduino-blink-generator
A simple C program that automatically generates Arduino LED blink (.ino) sketches.
-----------------------------------------------------------------------------------------------
A small C program that generates Arduino .ino files for blinking LEDs.

Features:
----------
Generates valid Arduino sketches;
User chooses LED pin;
User chooses blink delay;
Saves as .ino;
-----------
Built With:

C
File Handling
Strings
